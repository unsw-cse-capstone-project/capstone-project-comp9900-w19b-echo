Installation Guide

1. Download node v14.13.1 from below link and install
   https://nodejs.org/en/download/current/

2. Install Angular by below command
   npm install -g @angular/cli

3. Open it by Java IDE Intellij or Eclipse - The root folder is OLP.

4. To start front end, go to OLP/frontend folder in command/terminal and type "ng serve"
   The URL http://localhost:4200/ would be accessible.

5. Backend can be started in IDE as spring boot application
